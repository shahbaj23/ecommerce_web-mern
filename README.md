E-Commerce Website

Vercal: https://stylenest-front-end.vercel.app/

A full-stack E-commerce web application built using the MERN Stack (MongoDB, Express.js, React.js, Node.js).
The application allows users to browse products, search items, filter products, and manage their shopping experience efficiently.

Features
User Authentication (Login / Register)
Product Listing Page
Product Search Functionality
Product Filtering by Category
Sorting Products (Price Low → High / High → Low)
Product Details Page
Add to Cart
Responsive UI
REST API Integration
State Management using Context API / Redux Toolkit
